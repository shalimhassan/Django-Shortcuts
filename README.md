## Django Shortcuts
- ``` python
  cd folder_name = Enter into the folder (Change Directory)
  cd .. = 1 step back folder
  cd ..\.. = 2 step back folder
  cd ..\..\.. = 3 step back folder
  cls = Clear Screen
  code . = VS Code Open (Virtual Studio Code)
  mkdir = Make Directory (Create Folder)
  rd /s /q foldername = permanently folder delete
  ```
